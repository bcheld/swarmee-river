# opencode-compat pack

This pack adds a small system prompt section that nudges Swarmee River into an OpenCode-like workflow.

## Install (project-local)

```bash
swarmee pack install packs/opencode-compat
```

## Notes

- Packs are project-local via `.swarmee/settings.json`.
- Packs can provide tools (`packs/<name>/tools/*.py`) and SOPs (`packs/<name>/sops/*.sop.md`) as well.
